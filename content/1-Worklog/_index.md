**On this page**, you will need to introduce your worklog. **How** did you complete it? How many weeks did you take to complete the program? **What** did you do in those weeks?

Typically, and as a standard, a worklog is carried out over about 3 months (throughout the internship period) with weekly contents as follows:

**Week 1:** [Getting acquainted with AWS and basic services in AWS](1.1-week1/)

**Week 2:** [Learn basic services in AWS 2](1.2-week2/)

**Week 3:** [Learn basic services in AWS 3](1.3-week3/)

**Week 4:** [Learn basic services in AWS 4](1.4-week4/)

**Week 5:** [Attend the event and start the first steps of the sales web project](1.5-week5/)

**Week 6:** [Learn AWS services and gradually connect with the sales web project](1.6-week6/)

**Week 7:** [Learn AWS services and review mid-term](1.7-week7/)

**Week 8:** [Learn more about AWS services and review practice](1.8-week8/)

**Week 9:** [Progress with great strides of the Sales Web Project](1.9-week9/)

**Week 10:** [Understand and apply AWS to the Sales Web Project](1.10-week10/)

**Week 11:** [Understand and apply AWS to the Sales Web Project 2](1.11-week11/)

**Week 12:** [Understand and apply AWS to the Sales Web Project 3](1.12-week12/)
